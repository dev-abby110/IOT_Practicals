## Requirements

### Hardware
- Raspberry Pi 5 Model B
- ESP32 Development Board
- DHT11/DHT22 Sensors
- XBee Modules
- LoRa Modules
- HC-05 Bluetooth Module
- Various connecting wires and components

### Software
- Arduino IDE
- Node-RED
- Mosquitto MQTT Broker
- Python 3.x
- ThingSpeak Account
- Google Cloud Account
